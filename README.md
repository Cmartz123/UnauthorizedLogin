# Unauthorized Login Investigation Lab

## Objective

This project simulates a brute force login attack against a Windows workstation.

The objective is to investigate Windows Security Event Logs using Splunk to identify unauthorized login attempts, determine whether access was successful, and document the findings.

- Splunk
- Sysmon
